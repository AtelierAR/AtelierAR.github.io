Cree un repositorio en Github para los archivos a compartir
El archivo índice fue generado en Visual Code y es la base de todo lo que se visualiza la página
Para visualizar una imagen o video, se adjunta la dirección
Puede ser un vinculo de internet o puede ser una imágen almacenada en la computadora
Una vez hecho el tronco, se pueden crear ramas que permitan modificaciones a la página
El objetivo de de una rama es proponer adiciones
Todo esto de manera paralela hasta el visto bueno y anexarlo a main
En caso de detectar un problema se puede crear un issue dentro del repositorio
Hasta que quede resuelto, el issue permanecera abierto
